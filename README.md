# Raj Sticky Box

A powerful Elementor widget that creates stunning scroll-animated sticky cards with smooth reveal animations and a beautiful stacked card effect.

## Description

**Raj Sticky Box** is a premium-quality Elementor widget designed to showcase step-by-step processes, timelines, feature highlights, and more with an engaging scroll-triggered animation effect. As users scroll through your page, cards elegantly reveal themselves with smooth transitions, creating an interactive and visually appealing experience.

### Perfect For:
- How it works sections
- Process timelines
- Service showcases
- Feature highlights
- Step-by-step guides
- Onboarding flows

## Features

### Core Features
- **Scroll-Triggered Animations** - Cards smoothly reveal as users scroll with blur, fade, and scale effects
- **Sticky Positioning** - Cards stay fixed in view while scrolling for maximum impact
- **Stacked Card Effect** - Beautiful shadow layers behind the current card create depth
- **Unlimited Cards** - Add as many steps/cards as needed using the repeater field

### Customization Options

#### Card Settings
- Background color
- Border radius
- Padding (responsive)
- Box shadow
- Cards gap spacing
- Sticky top offset

#### Typography & Colors
- **Title** - Full typography controls (font, size, weight, style, color)
- **Description** - Full typography controls
- **Step Number** - Full typography controls with custom color

#### Icon Settings
- Icon picker (FontAwesome & custom SVG support)
- Icon color
- Icon background color
- Icon size
- Icon box size
- Icon box border radius

### Additional Features
- **Fully Responsive** - Customize settings for desktop, tablet, and mobile
- **Reset Styles Button** - One-click reset to restore default styling
- **Mobile Optimized** - Automatically switches to static layout on smaller screens
- **Lightweight** - No external dependencies, minimal footprint
- **Translation Ready** - Full internationalization support

## Installation

### From WordPress Admin
1. Download the plugin zip file
2. Go to **Plugins > Add New > Upload Plugin**
3. Choose the zip file and click **Install Now**
4. Activate the plugin

### Manual Installation
1. Download and extract the plugin
2. Upload the `raj-sticky-box` folder to `/wp-content/plugins/`
3. Activate through the **Plugins** menu in WordPress

## Requirements

- WordPress 5.0 or higher
- Elementor 3.0 or higher
- PHP 7.4 or higher

## Usage

1. Open any page/post with Elementor
2. Search for **"Raj Sticky Box"** in the widgets panel
3. Drag the widget to your desired section
4. Add your cards using the repeater:
   - Step Number (e.g., 01, 02, 03)
   - Icon
   - Title
   - Description
5. Customize the styling in the **Style** tab
6. Save and preview!
